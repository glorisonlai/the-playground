This portfolio, and all related servers are hosted on AWS. So, here are some ground rules:

  1. Do NOT attempt any excessive bruteforcing (Directory enum, Password spraying, etc). It is not needed, and will get you banned.

  2. All the source code (minus the flags) are stored on Github. You may find it useful to have a look.

  3. If you're doing a challenge, be nice! I've tried to make it impossible for any major disasters to happen, but no promises.

  4. If you find any unintended solutions or bugs, please contact me at lai.glorison@gmail.com

  5. Flags will only be in this format: FLAG{1H4V3R34D4L1T3RM54NDC0ND1T10N5}

  6. All code is free to be reused (God knows I have), and any contributions are welcome!


Since all challenges are web based (so far!), Burp suite, or any other proxy tool, is highly recommended.

There will be more challenges in future, so stay tuned.

Have fun!
